Sistema de Gestión Administrativa para el control de productos a través de código QR en Rey Mueblería

Aplicación web desarrollada con arquitectura de tres capas: interfaz de usuario, lógica de negocio y base de datos. Está orientada a optimizar el control de inventario mediante el uso de códigos QR únicos para cada producto.
El sistema permite:
- 🏷️ Generar códigos QR para identificar productos de forma rápida y segura.
- 📲 Registrar entradas y salidas de productos en tiempo real.
- 📁 Administrar el inventario con trazabilidad completa de cada movimiento.
- 👤 Gestionar usuarios y permisos para operar el sistema de forma controlada.
Diseñado para entornos que requieren eficiencia, orden y control logístico, esta solución se adapta a distintos tipos de organizaciones que manejan productos físicos y necesitan visibilidad sobre su flujo operativo

API REST con Spring Boot Seguridad, Validaciones y Queries Personalizadas

Este proyecto es una API REST construida con Spring Boot que integra:
- 🔑 Autenticación y autorización mediante Spring Security y JWT.
- 👥 Manejo de roles y control de acceso.
- ⚠️ Manejo global de excepciones para respuestas consistentes.
- ✅ Validaciones de datos usando Bean Validation y reglas personalizadas.
- 📊 Consultas personalizadas con JPA/Hibernate para demostrar dominio en repositorios y acceso a datos.
Diseñada para entornos que requieren seguridad, escalabilidad y buenas prácticas en el desarrollo de APIs modernas.
